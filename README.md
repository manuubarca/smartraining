# smartraining

Página inspirada en la aplicación "Quiero Jugar", donde se pueden gestionar distintos negocios del ambito del deporte.

Basandome en la app antes nombrada desarrolle esta aplicación para que el gimnasio donde trabajo tenga un sitio más personalizado, ya que en la app "Quiero Jugar"
estan asociados infinidad de gimnasios/canchas, etc.

En ella los socios podrían registrarse a través del formulario en la página de registro, obtener el 100% de la información necesaria para conocer
el negocio, reservar sus clases para asistir de acuerdo a horarios y cupos, y obtener contacto con el personal en caso de necesitar ayuda.
